## Getting Started

### Prerequisites

* npm

```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo

```sh
git clone https://github.com/SmirnovMaxim/react-burger
```

2. Install NPM packages

```sh
npm install
```

3. Run app

```sh
npm run start
```

### Testing

* Unit tests

```sh
npm run test
```

* Cypress tests

```sh
npm run cypress:open
```

### Deploy
```shell
npm run deploy-script
```
